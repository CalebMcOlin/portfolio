Full Stack Engineer Intern at Floop

When: June 2021 - October 2021

Intern Spotlight: Caleb McOlin at Floop

Description of Work:

While at Floop Edu I was put in charge of the development of a brand-new feature that our team called “Student Engagement Metrics”. This new feature was designed to track the conversations between the students and the teacher. Doing this entailed calculating the number of total comments, the number of comments seen, and the number of comments replied to by each student for each and every assignment submission. New database collections, triggers, scripts, and queries were created for the backend while new UI elements were created to display the stored metrics data on the frontend. Once the feature was complete and approved a new single-use script was needed to run over the existing production database to modify and tune it to be compatible with the new metrics tracking.

Takeways:

  • I learned a great deal about NoSQL cloud functions and how to ensure they are created in a clean and efficient way. This goes hand-in-hand with the importance of Promises and making sure they are handled correctly.
  • My knowledge of React, React hooks, functional components and third party React frameworks skyrocketed during my time at Floop. I feel much more comfortable reading, navigating and writing code using React.

3D Software Engineer Intern at CodeDay

When: June 2021 - September 2021

See the Repo Here: GitHub

Description of Work

During my time with CodeDay I worked collaboratively with other forward-thinking 3D engineers. Our project was to create a “3D Model World”. This 3D environment would allow users to upload their own 3D models -be that from photogrammetry or otherwise created, into a real-world environment using a map API. The user would then be able to name, move, re-size, and rotate their models to their liking. Once a user had created a map instance, they would then be able to share the URL with others, in order to show off all of their models.

Contribution:

  • I was in charge of the 3D technology. This meant learning the tools well enough to not only create the needed assets myself but also be knowledgeable enough to explain and teach them to partners.
  • I was the lead for the GitHub repo management. This entailed ensuring all PR merges, code reviews, and repo permissions were in place correctly. as well as keeping the readme up to date.

About This Website

I decided early on not to use popular libraries like React.js or frameworks like Angular.js in order to expand my knowledge, comfortability, and capability with the fundamentals. As such this website was built from scratch utilizing the core features of CSS3, HTML5, and JavaScript. The 3D environment present in the background was built with Three.js and is aided by 3D libraries such as Gsap.js and Tween.js. On the development site, VSCode was my primary editor while I used Node.js, NPM, and Vite.js for building and testing. The deployment and live hosting of this website is handled by AWS.
